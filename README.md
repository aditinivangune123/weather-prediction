# 🌦️ Weather Prediction Dashboard

An AI-powered Weather Prediction System built using Machine Learning and Streamlit.  
This project predicts whether the weather will be **Sunny ☀️ or Rainy 🌧️** based on input data.

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/aditi-nivangune/weather-prediction.git
cd weather-prediction
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run weather.py
```
